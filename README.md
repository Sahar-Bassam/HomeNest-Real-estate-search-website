# HomeNest

HomeNest is a full-stack Laravel web application designed to help users browse and explore real estate properties. It provides a clean interface for viewing property listings, detailed pages, and managing data through a structured backend.

---

## Demo

Watch the full demo here:

---

##  Features

*  User authentication (Login system)
*  Browse all available properties
*  View detailed property information
*  Property categorization
*  Pagination for listings
*  Delete properties
*  Clean and minimal UI design

---

## Tech Stack

* **Backend:** Laravel (PHP)
* **Frontend:** Blade Templates
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **Database:** MySQL

---

## Key Functionality

* Dynamic property listing using Laravel collections
* Relationship handling (Property → Category)
* Server-side validation with error handling
* RESTful routing (CRUD operations)
* Pagination using Laravel built-in methods
